Tecnologías

Lenguajes de la Web

La base de cualquier página. HTML estructura el contenido, CSS le da estilo y JavaScript añade interacción. Juntos crean experiencias visuales, responsivas e interactivas.

HTML: la estructura de la web

HTML (HyperText Markup Language) es el lenguaje que da forma a cualquier página web. Define la estructura del contenido mediante etiquetas que organizan textos, imágenes, enlaces, tablas o vídeos. Es como el esqueleto de un sitio: sin él, no existiría la base sobre la que construir diseño o interacción.

CSS: el estilo y la apariencia

CSS (Cascading Style Sheets) se encarga del aspecto visual de las páginas. Permite definir colores, fuentes, márgenes, fondos, animaciones y distribución de los elementos. Gracias al CSS, una web puede adaptarse a distintos dispositivos mediante el diseño responsive y adquirir personalidad propia con estilos modernos y coherentes.

JavaScript: la interacción

JavaScript añade vida y dinamismo a las páginas web. Con este lenguaje, los elementos pueden responder a acciones del usuario, actualizar datos sin recargar la página o mostrar animaciones y efectos. Es esencial para crear experiencias interactivas como formularios inteligentes, menús dinámicos o juegos en línea.

Lenguajes de datos

Para que las aplicaciones se comuniquen, se usan formatos de intercambio como JSON y XML. Las APIs permiten enviar y recibir datos entre servicios (por ejemplo, mapas, clima o redes sociales).

Lenguaje humano digital

En redes usamos símbolos y abreviaturas para comunicarnos rápido: emojis, memes, expresiones como LOL, BRB o BTW. La netiqueta reúne normas básicas de respeto y claridad al comunicarnos online.
