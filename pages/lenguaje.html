Tipos de Lenguaje

Lenguajes de la Web

La base de cualquier página. HTML estructura el contenido, CSS le da estilo y JavaScript añade interacción. Juntos crean experiencias visuales, responsivas e interactivas.

Lenguajes de datos

Para que las aplicaciones se comuniquen, se usan formatos de intercambio como JSON y XML. Las APIs permiten enviar y recibir datos entre servicios (por ejemplo, mapas, clima o redes sociales).

Lenguaje humano digital

En redes usamos símbolos y abreviaturas para comunicarnos rápido: emojis, memes, expresiones como LOL, BRB o BTW. La netiqueta reúne normas básicas de respeto y claridad al comunicarnos online.
